えびす 概要 説明書
===================================
2015.09.09 mori@homoeopathy.co.jp

### 目的

 *    だれでもオンラインで買い物ができる
 *    適用書番号とパスワードを入力するとだれでも適用書レメディーを購入できる
 *    ホメオパスは適用書を入力できる
 *    HE/HJ/HPの代理店は
  *       対象商品を優待で購入できる
  *       エンドユーザに代行直送して当社から手数料を受け取ることができる
  *       自分の店の小売伝票を立てることができる
 *        会員登録手続きすると応分のポイントを受け取ることができる

 *    ※当社の従業員は
  *       上記にまつわる手続きの結果を修正することができる
  *       実店舗において売上を立てることができる
  *       発送所において発送伝票を印字することができる
  *       請求書を発行することができる

### 対象
 * 通りすがりの人（初めての訪問者）は顧客登録してから買い物してもいいし、顧客登録なしでも買い物ができる
 * 登録済みの人はログインして買い物ができる

### 決済方法
 * 支払い方法は「代引き」を原則とする
 * ただし、TY,HP,HE,HJ各社と契約した取引先は「月末締め、銀行振込」を原則とする
 * ただし、TY,HP,HE,HJ各社と契約した取引先は「代行直送」を選択することもできる
 * また、TYモールに登録した会員は「代引き」「銀行振込」から選択できる

### オンライン注文の手順
 1.   商品を「かご」に入れる
 2.   「かご」を見て
   *     金額や配送先を確認し、クリックすると注文が確定する
   *     ただし、配送先などの情報が不完全な場合、注文が確定できない


WEBページ概要
-------------------
### 固定ページ
*    つねに一定の文言を表示する。

*    TY,HP,HE,HJ、トミーローズ:　それぞれに固有のページを用意する。
 *    各社はそのページを自由にデザインして表示することができる
 *    ただし、ページのヘッダとフッタはTYモール共有のHTMLが適用される

### 商品ページ
 *    商品の情報を表示する。つねにデータベースの情報を参照し、表示する

### 適用書検索ページ
 *    適用書番号とパスワードを入力すると適用書が表示される
 *    ただし、一度「購入」された適用書は検索されないので繰り返し購入できない

### かごページ
 *    顧客が決定した商品を表示する
 *   顧客が入力した配送先を表示する
 *   注文の確定ボタンを表示する

### 履歴ページ
 *   ログインした顧客が自分の履歴を見るためのページ
 *   ログインしていない場合、表示されず、代わりにログイン画面へ遷移する

 *   顧客は
  *      自分の住所・氏名などを修正できる
  *      自分の注文履歴を見ることができる
  *      自分の住所以外の配送先を登録・修正することができる

### 会員登録ページ
 *   氏名、かな、住所、TELなどを入力すると、会員登録が完了する
 *   学生やとらのこ会など、グループ各社の顧客データを移行することもできる

### 問い合わせページ
 *   通りすがりの人、および登録済み顧客がメッセージを入力・送信することができる
 *   ログインした顧客に対しては本人のEmailを送信元に設定する

### 適用書ページ
 *   ホメオパスなら、適用書ページを開くことができる
 *   適用書を作成することができる


WEBページ詳細
-------------------
### 商品ページ
 *   商品の一覧
  *      以下、商品検索、各社トップ、カテゴリトップにて商品の一覧を表示する
  *      商品の一覧にあたっては
   *         画像あり段組み表示　１２件ずつ
   *         画像なし一列表示　　５０件ずつ　いずれかを選ぶことができる

 *   商品検索

        TY,HP,HE,HJ、TROSEにかかわらず、すべての商品を対象にキーワード検索することができる

 *   各社トップページ

        TY,HP,HE,HJ、TROSE各社のトップページにおいて各社の商品を一覧表示する

 *   カテゴリトップページ

        大区分、中区分のトップにおいて、そのカテゴリに属するすべての商品を表示する

 *  商品の紹介:
        ひとつの商品について紹介する

   *     商品画像（なし、または１枚または２枚以上）
   *     価格
   *     ポイント
   *     「かごに入れる」ボタン
   *     「お気に入りに追加」ボタン
   *    商品ポイント:
        データベースに登録された商品の状態と、ログインした顧客の状態に基づいて、各商品のポイントを計算し、表示する
   *    商品画像
```
        原画は商品コードと紐付けてJPEGサーバに保存する
        @common/images/<商品コード>.jpg
        2枚以上ある場合はファイル名に番号を付加する
        @common/images/<商品コード>.jpg 最初の画像
        @common/images/<商品コード>_01.jpg　次の画像
        @common/images/<商品コード>_02.jpg　その次の画像
        @common/images/<商品コード>_03.jpg
        原画は最大　1000　x　1000px　とする（変換にメモリを消費するので）

        顧客が実際に見る画像は　500x500px　を標準とする
        WEB用画像は随時レンダリングする
         @web/assets/images/<商品コード>.jpg
        
        サムネイルは別の名前で随時レンダリングする
        60pxの場合、thumb/60/に保存する
         @web/assets/images/thumb/60/<商品コード>.jpg
```
        
### かごページ
３種類に分かれる。

 *   TROSEカート 　　TROSEの商品のみ      
 *   TYカート　　　　 TYの商品のみ
 *   まとめ買いカート　それ以外の商品全部

  *    ログイン済みの場合、注文を確定することができる
  *  ログインまだの場合、本人Emailと配送先の入力が注文確定の前に必須となる

 * 注文の確定
            顧客が注文を確定したらメールを送信する。
```
            TO：顧客本人
            BCC：当社・発送部（shopping@toyouke.com, shopping@homoeopathy.co.jp）〓要確認〓
```

### 履歴ページ
 *    本人の情報        ※閲覧と修正ができる
  *       住所
  *       氏名
  *       性別（任意）
  *       生年月日（任意）
  *       電話
  *       Email
  *       メルマガ配信希望（初期値＝YES）

 *    配送先　住所録        ※閲覧と修正・削除ができる
  *        住所・氏名・電話

 *    注文の履歴
  *      年月日
  *      注文番号
  *      状態

 *   お気に入り        ※閲覧と削除ができる
  *      商品画像
  *      商品名

   1.     shop@toyouke.com  または　member@toyouke.com 〓要確認〓  にメールを送信する           
   2.     データベースにメール本文を記録する
