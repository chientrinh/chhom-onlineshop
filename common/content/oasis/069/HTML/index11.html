Oasis 9　とらこ先生　ファミリーホメオパスコースオープンキャンパスOasis No.69 Spring to Summer 2016し、酸性体質改善ができます。　塩を取るな、血圧が上がると教えられましたが、実際は塩がないから血圧が上がるのです。塩があれば血流はスムーズに流れるのです。圧が上がるということは流れないから血圧を上げてプッシュしているということです。圧が上がるというのは毛細血管まで血液を入れなければならない理由があるわけです。塩があれば血圧は下がるんです。精製されていない自然塩をとればの話です。　医学界を中心に減塩政策をしました。今なおこれが続いていま・日本人は悪い事をしたんだという罪悪感政策・農薬、化学肥料を推進する農業政策・予防接種義務化政策です。日本は憲法も学校教育も何から何まで全部政策していきました。その延長線上に精製塩を使ったり、減塩政策もあると思っています。精製塩を使わせたり、減塩させるのはなぜかというと、私たち日本人が、にがり分が入っているようなミネラルを使えば使うほど健康になるからです。日本人の血圧が高いのは塩を取り過ぎるからとまことしやかに洗脳し、保存料もみんな自然塩から化学品の精製塩にしてしまいました。それによって塩を取らないような日本人になっていきました。　塩がないと神経伝達が出来ません。伝達するには塩がいります。今、膵液を出せてと脳からの指令は電気信号です。電気信号を司るのは塩です。塩はミネラルが多い海塩を使い、焼くと弱アルカリになりますから、体を弱アルカリにす。沢庵の食べ過ぎだとかね。豊受オーガニクスレストランは必ずと言っていいほど三切れの沢庵か野沢菜を出します。食べてもらいたいからです。体には良い塩がいるからです。日本は保存食は塩のみでやっていました。塩分を抑えた結果、毒性の強い化学物質、保存料が入ってきました。更に保存性を高めるためにp H 調整剤が入りました。減塩すると味が薄くなるのでおいしくなくなるので、仕様がないからグルタミン酸ナトリウム化学調味料を入れなくてはならなくなりました。自然塩があれば体がミネラルを求めているのでおいしいと感じるのに…。また酸化して食品の色がおかしくなるので、酸化防止剤も入れたり、着色料も入れるようになりました。現に１５００ほどの食品添加物が日本では使われて、二位アメリカでさえ百八十個、英国はたったの十二個しか許可が下りていません。こうして一貫して日本人を弱体化させる政策をとり、歯向かわないようにしているのではないかと思います。健康であればあるほど事実を言うだろうし、行動も起こすでしょう。体が弱ければ行動は起こせません。＊動け！ ミトコンドリア　今、私たちはこの体に何を取り戻さなければいけないか考えましょう。自分には自己治癒力があるんだということ。自分は自然なものを食べていれば、必ずやそれが身となり、骨となり、健康を作ってくれるんだということを。心根はいろんなことがあっても恐れない。恨まない。憎まない、平常心でいてストレスにならない。安保先生が言うところの「解糖系」を使うアドレナリン型ではなくてミトコンドリア型。リラックスして、そして炭水化物からゆっくりエネルギー生産をする形。ミトコンドリアが動けばそんなに多量の岩塩海塩水分0.02 7.56不溶解分0.01 0.01NaCl（塩化ナトリウム） 99.8 97.4CaSO4（硫酸カルシウム） 0.14 0.99MgCl2（塩化マグネシウム） 0 0.73MgSO4（硫酸マグネシウム） 0 0.52CaCl2（塩化カルシウム） 0 0KCl（塩化カリウム） 0 0.16Na2SO4（硫酸ナトリウム） 0 0にがり分0 1.41Fe(鉄ppm) 3.5 7.9岩塩と海塩の分析例（ドライベース%）岩塩と海塩
